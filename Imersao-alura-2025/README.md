# 🚗 Body Kits Showcase — Liberty Walk & Rocket Bunny

Um projeto desenvolvido durante a Imersão Alura 2025, com o objetivo de apresentar de forma mais moderna, organizada e visualmente atraente os famosos body kits das marcas **Liberty Walk** e **Rocket Bunny**. 

O projeto utiliza **HTML, CSS e JavaScript**, consumindo dados a partir de um arquivo **JSON**, e está hospedado no **GitHub Pages**.

## 📘 Sobre o Projeto

Os sites oficiais da **Rocket Bunny** e **Liberty Walk** apresentam suas linhas de body kits, mas de forma pouco prática e, no caso da Rocket Bunny, com um layout visual limitado.

Além disso, a Liberty Walk redireciona o usuário para outra página para visualizar detalhes de cada produto, quebrando a experiência de navegação.

Este projeto foi criado para **melhorar a experiência do usuário**, reunindo:

- Uma **interface moderna** e responsiva  
- Uma navegação fluida entre marcas  
- Uma exibição clara dos body kits  
- Um **modal completo** com:
  - Nome do carro  
  - Marca  
  - Parts list do body kit  
  - Carrossel de imagens  

Tudo isso **sem recarregar a página**.

## 🚀 Funcionalidades

### ✔ Alternância entre marcas  
Permite navegar entre Liberty Walk e Rocket Bunny de forma direta e intuitiva.

### ✔ Sistema de busca  
Filtre carros por nome, modelo ou título do body kit.

### ✔ Cards de exibição  
Cada body kit possui:
- Imagem de destaque  
- Nome do kit  
- Veículo base  
- Marca correspondente  

### ✔ Modal informativo com carrossel  
Ao clicar em qualquer card, um modal é aberto contendo:
- Carrossel de imagens em alta qualidade  
- Nome do carro  
- Marca  
- Partes do body kit instaladas  
- Link para o site oficial  

### ✔ Layout moderno inspirado no estilo das marcas  
Interface baseada na estética clássica de ambas as fabricantes.
